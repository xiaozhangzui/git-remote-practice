项目名称：git-remote-practice

本任务的练习目标：
学习 Git 远程仓库的基本使用方法，包括创建远程仓库、关联本地与远程仓库、推送本地提交到远程、从远程拉取更新等操作，初步掌握本地与远程仓库协作的工作流程。

姓名/学号：张小帅

远程仓库使用记录：
- 使用的远程平台名称：GitHub（github.com）
- 远程仓库名称：git-remote-practice
- 远程仓库的主要作用：用于备份本地代码、实现多设备之间的代码同步和团队协作开发。
- 同步过程中遇到的问题或注意事项：首次推送时需要使用 git push -u origin master 设置上游分支；推送需要通过 Personal Access Token 进行身份验证，不能使用密码登录；确保远程仓库为空再首次推送，否则可能产生冲突。
